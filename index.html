About Me

코드와 디자인의 밸런스를 중요하게 생각하는
웹 퍼블리셔 김준수입니다 🙇🏻‍♂️

저는 코드와 디자인 간의 균형을 잃지 않는 방법을 잘 이해하고 있습니다.

디자인 요소를 고려하며 웹 페이지의 기능과 성능을 향상시키기 위해 향상 최적의 코드를 작성합니다.

효율적인 알고리즘을 사용하여 웹 페이지의 성능을 향상시키고, 웹 페이지의 로딩 속도를 최적화하기 위해 코드를 최소화합니다.

또, 웹 표준과 웹 접근성을 준수하여 사용자들이 웹 페이지를 쉽게 이용할 수 있도록 합니다.

Project

Project1

L/DERS

SCSS를 활용하여 작업한 웹사이트입니다. SCSS를 공부하면서 기존 CSS보다 작성하기 편리하고 여러 구성에 대한 효율적인 관리 등 좋은 장점들을 알게 됐습니다. 특히 반응형 제작 시 기존 CSS 미디어쿼리 작성보다 직관적으로 만들 수 있었던 것이 장점으로 느껴졌습니다.

  • 기 술HTML, CSS, SCSS, JavaScript, jQuery
  • 호 환 성
  • 기 간2일
  • 기여도메인페이지 제작 100%
구현화면 깃허브
코드를 nav, header, main, footer로 구조화 하고 scss로 스타일 작업을 진행하였습니다.
nav에 toggle 버튼을 넣어 숨겨진 메뉴를 보여주고 다시 숨길 수 있게 하였습니다.
main은 section으로 나누었고 핵심 콘텐츠인 모델 소개 부분에 슬라이더 기능도 추가하였습니다.
전체적으로 페이지 요소가 나타날 때 fade in 효과도 추가하였습니다.

Project2

빙그레

코딩 수업을 배우고 스스로 처음 진행해 본 빙그레 프로젝트 입니다. 와이어 프레임 작업을 진행 후 구조를 파악하고 CSS 를 적용하여 웹 사이트의 디자인을 꾸며주고 Javascript, jQuery 를 적용해서 인터랙티브한 기능을 추가하였습니다. 빙그레 사이트는 저의 코딩 실력을 향상 시키고 스스로 구조를 파악할 수 있는 방법과 화면을 구현해볼 수 있는 기회를 제공해주었습니다.

  • 기 술HTML, CSS, JavaScript, jQuery
  • 호 환 성
  • 기 간3주
  • 기여도메인페이지 100% 제작 & 웹 접근성 적용과 플러그인 미사용
구현화면 깃허브
nav 부분의 gnb 메뉴를 2depth로 구현하였고, main을 섹션별로 나누어 전체 구조를 작성하였습니다.
핵심 콘텐츠인 제품 이미지 부분에 슬라이더 기능도 추가하여 동적인 효과를 주었습니다.
